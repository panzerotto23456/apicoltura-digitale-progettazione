## KY-038 (Sensore Sonoro)

| Intensità | Stato dell'Arnia | Soglia Sonora | Comunicazione al Server |

| **Silenzio** | **Inattività / Glomere** | 20 - 100 Hz | Messaggio di "Quiescenza". |

| **Basso** | **Attività di Routine** | 100 - 150 Hz | Messaggio di "Stato Normale" (colonia sana). |

| **Medio** | **Ventilazione** | 150 - 250 Hz | Allarme "Forte Attività": deumidificazione o calore. |

| **Elevato** | **Pre-Sciamatura** | 250 - 350 Hz | **AVVISO ATTENZIONE**: Ronzio potente pre-sciame. |

| **Assente** | **Anomalia / Morte** | 0 - 20 Hz | **ALLARME**: Assenza vibrazioni (guasto o perdita). |
